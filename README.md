# Прогнозирование прогрессирования глаукомы с использованием глубоких нейронных сетей

## Описание проекта
Данный проект представляет собой реализацию модели глубокого обучения для прогнозирования прогрессирования глаукомы на основе медицинских данных.

## Установка
1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование
1. Убедитесь, что файл с данными `glaucoma_dataset.csv` находится в корневой директории проекта
2. Запустите основной скрипт:
```bash
python glaucoma_prediction.py
```

## Структура проекта
- `glaucoma_prediction.py` - основной скрипт с реализацией модели
- `requirements.txt` - файл с зависимостями
- `glaucoma_dataset.csv` - набор данных
- `training_results.png` - графики результатов обучения (генерируется после запуска)

## Модель
Модель представляет собой глубокую нейронную сеть с тремя скрытыми слоями и регуляризацией через Dropout. Используется бинарная классификация для предсказания прогрессирования глаукомы.

## Метрики
- Accuracy (Точность)
- AUC (Area Under Curve)
